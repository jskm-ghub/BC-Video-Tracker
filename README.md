# BC-Video-Tracker
Application for cataloging BC's video collection

Video Cataloging System – a Java application that indexes and tracks video files stored across multiple external drives for the school marketing department.

Features:
     - Detects external drives
     - Scans file structures
     - Updates MySQL database
     - Tracks which files are located on which drives
     - Has a simple UI for user interaction
     - Provides file search
